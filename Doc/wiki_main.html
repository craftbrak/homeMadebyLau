# Description du projet Le site internet "Homemade by Lau" est un site de cuisine destiné à Laurence Costermans. Le but premier est de pouvoir y poster ses recettes de cuisine, ainsi que de diriger son audience vers les différents produits (ustensiles ou ingrédients) utilisés dans les recettes présentées. Dans un second temps, il doit également lui permettre de promouvoir et proposer des ateliers de cuisine en français et en anglais, et d'en orchestrer l'organisation sur la plateforme # Description du client Laurence Costermans est professeure d'anglais et conseillère Guy Demarle. Par le biais de ce site internet, elle cherche à promouvoir les produits Guy Demarle via son lien personnel, afin d'être rénumérée s'ils achètent un produit Guy Demarle par la suite.
